This page defines icons from smallest to largest.

If the favicon reads "ROOT ICO", your browser loaded /favicon.ico.

If the favicon reads "ROOT PNG", your browser loaded /favicon.png.

If the favicon reads "16 ICO", your browser loaded /icon16.ico.

If the favicon reads "16 PNG", your browser loaded /icon16.png.

If the favicon is simply a number like "32", your browser loaded /icon32.png, etc.
